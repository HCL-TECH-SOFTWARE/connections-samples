Visual Update 1 for HCL Connections

Overview

This application uses Connections Customizer to apply updated styling to HCL Connections. Customizer layers this new design directly on top of the existing Connections. The result is a modern, airy, and less cluttered visual appearance.

Options for using Visual Update 1

Visual Update 1 not only delivers an improved visual design but also serves as a basis for you to make your own visual updates, such as change to use your company colors.

  • Apply Visual Update 1 to Connections

    Cloud: Install from App Catalog

    Cloud customers can install the Visual Update 1 for HCL Connections application from the app catalog in only a few clicks.

    On-premises: Create Application

    On-premises customers can download these files, unchanged, and create a Customizer application in their registry.

  • Customize based on Visual Update 1

    Learn how to Customize

    Both Cloud and On-premises customers can download these files and use them as a basis to further customize Connections. For example, you can apply the new visual styles and also update the brand colors used.
